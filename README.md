📝 Description : A modern full-stack authentication system built with Next.js and Express.js. It provides secure user authentication, email verification, password reset functionality, and JWT-based authorization with HTTP-only cookies. The project focuses on security, clean architecture, and a smooth user experience.



🚀 Tech Stack

Frontend: 
Next.js - React - TypeScript - Tailwind CSS - Zustand - Axios - Framer Motion - React Hot Toast - React Icons - Day.js

Backend: 
Node.js - Express.js - MongoDB - Mongoose - JWT (jsonwebtoken) - bcryptjs - Cookie Parser - CORS - Dotenv - CryptoJS - Resend API

✨ Features
- User Registration
- Secure Login & Logout
- Email Verification
- JWT Authentication
- Protected Routes
- Forgot Password
- Reset Password
- HTTP-only Cookie Authentication
- Password Hashing using bcrypt
- Secure Token Generation
- MongoDB Database Integration
- Responsive UI
- Toast Notifications
- Smooth Animations
- State Management with Zustand
- API Integration using Axios
